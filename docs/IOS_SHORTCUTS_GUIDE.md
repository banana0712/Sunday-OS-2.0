## 🎤 快捷指令一：Sunday 主入口（文字+语音双模式）

这是最常用的指令——一键对话 Sunday。

### 创建步骤

1. 打开 iPhone **「快捷指令」** App
2. 点右上角 **「+」** 新建
3. 点顶部标题改为 **「Sunday」**
4. 按以下顺序添加操作：

#### 第 1 步：获取输入

```
操作: 「听写文本」
  - 语言: 中文（普通话 - 中国大陆）
  - 停止聆听: 轻点后
```

#### 第 2 步：构建 API 请求

```
操作: 「获取 URL 内容」
  - URL: https://sunday-os-20-278970-7-1451274775.sh.run.tcloudbase.com/api/chat
  - 方法: POST
  - 请求体: JSON
```

**请求头**（点「显示更多」→「头部」）：
| 头部名称 | 值 |
|----------|-----|
| `Content-Type` | `application/json` |
| `X-API-Key` | `sunday-os-dev-key-change-in-production` |

**请求体 JSON**：
```json
{
  "message": "{听写的文本}",
  "session_id": "iphone-daily"
}
```

> ⚠️ 把 `{听写的文本}` 替换为「选择魔法变量」→ 选择「听写的文本」

#### 第 3 步：解析结果

```
操作: 「从输入获取词典值」
  - 输入: 「URL 的内容」
  - 键: reply
```

#### 第 4 步：朗读结果

```
操作: 「朗读文本」
  - 文本: 「词典值」
  - 语言: 中文
```

#### 第 5 步：显示结果

```
操作: 「显示结果」
  - 文本: 「词典值」
```

### 💡 进阶版：文字优先，语音兜底

如果不想每次都语音输入，可以这样改：

1. 删除「听写文本」
2. 添加：

```
操作: 「从菜单中选择」
  - 选项: ["文字输入", "语音输入"]
```

然后在每个菜单项下分别添加对应逻辑：
- **文字输入** → 「要求输入文本」
- **语音输入** → 「听写文本」

## 🌅 快捷指令二：早安 Sunday（每日简报）

每天早晨自动获取个性化简报。

### 创建步骤

1. 新建快捷指令，命名为 **「早安 Sunday」**
2. 添加以下操作：

```
操作: 「获取 URL 内容」
  - URL: https://sunday-os-20-278970-7-1451274775.sh.run.tcloudbase.com/api/chat
  - 方法: POST
  - 头部:
    Content-Type: application/json
    X-API-Key: sunday-os-dev-key-change-in-production
  - 请求体 JSON:
    {
      "message": "早安！给我今天的简报，包括日期、天气建议、我可能有的日程、以及一句鼓励的话。",
      "session_id": "iphone-brief"
    }
```

```
操作: 「从输入获取词典值」→ 键: reply
```

```
操作: 「朗读文本」→ 文本: 词典值
```

```
操作: 「显示结果」→ 文本: 词典值
```

### ⏰ 设置自动化

1. 切到 **「自动化」** 标签页
2. 点 **「+」→「创建个人自动化」**
3. 选择 **「特定时间」**
4. 设为每天早上 7:30
5. 添加操作 **「运行快捷指令」** → 选择「早安 Sunday」

## 🔗 添加到主屏幕 / 锁屏

### 方法一：添加到主屏幕

1. 长按快捷指令 → **「详细信息」**
2. 点 **「添加到主屏幕」**
3. 选图标和名称

### 方法二：锁屏小组件（iOS 16+）

1. 锁屏界面长按 → **「自定义」**
2. 添加小组件 → 搜索「快捷指令」
3. 选择「Sunday」快捷指令

### 方法三：操作按钮（iPhone 15 Pro / 16 Pro）

1. **设置 → 操作按钮**
2. 滑动到「快捷指令」
3. 选择「Sunday」快捷指令
4. 以后长按侧边按钮就能唤醒 Sunday！

### 方法四：Hey Siri 唤醒

直接对 iPhone 说：
> **"Hey Siri, Sunday"**

Siri 会自动运行「Sunday」快捷指令，然后你就能语音对话了。

## ⚡ 进阶技巧

### 1. 个性化 session_id

不同的 session_id 让 Sunday 在不同场景下保持独立的对话上下文：

| 场景 | session_id |
|------|-----------|
| 日常聊天 | `iphone-daily` |
| 工作 | `iphone-work` |
| 个人日记 | `iphone-journal` |
| 学习 | `iphone-learn` |

### 2. 自动粘贴到剪贴板

在快捷指令末尾添加：
```
操作: 「拷贝到剪贴板」
  - 内容: 词典值（reply）
```

这样你可以直接粘贴回复到其他地方。

### 3. 流式响应（高级）

如果你想要打字机效果，把 URL 换成：
```
https://sunday-os-20-278970-7-1451274775.sh.run.tcloudbase.com/api/chat/stream
```

但快捷指令对 SSE 的支持有限，推荐用非流式接口。

## 🛠️ 故障排查

| 问题 | 解决 |
|------|------|
| "无法连接到服务器" | 检查网络，确认 API 地址以 https:// 开头 |
| "Unauthorized" | 检查 X-API-Key 是否正确 |
| 返回空白 | 确认请求体 JSON 格式正确，message 字段不为空 |
| 回复太慢 | 正常，蚂蚁百灵模型响应约 2-5 秒 |
| 听写不识别 | 设置 → 通用 → 键盘 → 启用听写 |
